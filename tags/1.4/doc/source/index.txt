*******
Welcome
*******

PySAL is an `open source <http://code.google.com/p/pysal/>`_  cross-platform library of spatial analysis functions written in
Python. It is intended to support the development of high level applications
for spatial analysis.


****************
Getting Involved
****************

Prospective developers can learn more in the :doc:`developer guide
</developers/index>`.

You can track PySAL development, submit bug reports, and make feature requests 
`here <http://code.google.com/p/pysal/issues/list>`_. 

Downloads are `here <http://code.google.com/p/pysal/downloads/list>`_.
